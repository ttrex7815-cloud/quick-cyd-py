# Python Hello World Codespace
 
A quick repo for a simple Python setup in GitHub Codespaces.

## Getting Started

### 1. Create a Codespace
- Click the **"Code"** button at the top of this repository on GitHub.
- Select **"Create codespace on main"**.
- Wait for the Codespace to initialize (this may take a minute).

### 2. Install the Recommended Extension
- When prompted, install the **Python** extension (ms-python.python).
- If not prompted, open the Extensions sidebar (left bar) and search for "Python" by Microsoft, then install it.

### 3. Run the Hello World Program
- Open the `hello.py` file in the editor.
- Click the **Run** button in the top right, or right-click in the editor and select **Run Python File in Terminal**.
- You should see `Hello, world!` printed in the terminal.

### 4. Debugging (Optional)
- With `hello.py` open, click the **Run and Debug** icon on the left sidebar.
- Select **Python: Current File** and click **Start Debugging**.
